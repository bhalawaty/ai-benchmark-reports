How Tokens Work — Simple Explanation

Tokens are the "currency" of AI models. Think of them like data credits — every word, number, and symbol costs tokens.

💬
User Asks a Question
"Show me employees who were late this month"
~50 tokens
🔧
System Adds Context
Database schema, rules, instructions sent with every request
~2,500 tokens
🤖
AI Returns SQL + Explanation
SQL query + brief explanation of results
~1,100 tokens

One Request = How Many Tokens?

Range depends on model. Gemini models tend to produce longer outputs (~4,500+ tokens total). GPT 4.1 and Claude are more concise (~2,500–2,800 total).

Why the system prompt is included every time: Unlike a chat where previous context is "remembered," each AI SQL request sends the full database schema and rules with the question. This is ~2,500 tokens of input on every single request — it's the biggest cost driver.

Token Subscription Model — How It Works

Each company subscribes to a token quota — like buying a data plan. They don't pay per-month, they pay for a bucket of tokens that gets consumed as employees ask questions.

💳
Company Buys Tokens
e.g., "1M token plan"
Like buying a prepaid data pack
👥
Employees Use Chat
Each question uses ~2,500–4,800 tokens
Shared across all admins
🚫
Quota Runs Out
Company can top up or upgrade
We set limits per plan tier
Why token-based (not monthly)? A company with 5 admins asking 10 questions/day uses very differently from one with 50 admins asking 100 questions/day. Token quotas let us charge fairly based on actual usage — like cloud bandwidth or mobile data.

Pricing Strategy Insights

Best for SaaS
Gemini 2.5 Flash
A 1M token plan costs us less than $1. Even a generous 5M plan costs only ~$5. This gives the widest margin for subscription pricing — we could charge $10–50/plan and have excellent margins.
Premium Only
Claude Opus 4.6
A 1M plan costs us ~$11. A 5M plan = ~$55. This model is only viable for premium enterprise tiers or internal use — not volume SaaS at our current pricing.
Recommendation: Use Gemini 2.5 Flash (or fine-tuned Flash) as the default model for all companies. Offer Gemini 2.5 Pro as a "Pro AI" upgrade tier. Reserve Claude Opus for internal team use only.
