What We Tested

We asked 7 AI models to answer 7 real HR questions — each in both English and Arabic (14 tests per model, 98 tests total). Questions range from simple ("How many employees per department?") to complex ("Salary breakdown per department").

Each answer was scored on: Did it get the right data? and Did it respond in the correct language? For our Saudi market, Arabic support is critical — if a user asks in Arabic, everything must come back in Arabic.

Why these 5 models?
Out of 7 models tested, 2 were eliminated because they cannot respond properly in Arabic. Gemma 3 returned English for every Arabic question. Qwen 3 235B returned English for 4 of 7 Arabic questions. The remaining 5 are the viable options.

📌 MoE Architecture Explained
MiniMax M2.5 uses a Mixture-of-Experts (MoE) architecture: 230 billion total parameters but only ~10 billion are active per token. This means the full model requires large GPU memory (4x A100 80GB minimum) but inference speed is competitive because only a fraction of parameters process each token.

⚠️ Self-Hosting: 4× More Expensive Than Qwen 3 32B
MiniMax M2.5 requires 4x A100 80GB GPUs (~$4,351/mo) vs Qwen 3 32B's 1x A100 80GB (~$1,088/mo). That's 4× the monthly cost for self-hosting. Unless your query volume is massive, we recommend using MiniMax via API and reserving self-hosting for Qwen 3 32B where the economics are better.
💡 When to choose MiniMax M2.5?
Choose MiniMax M2.5 via API if you want near-top accuracy (87/100) with zero infrastructure. At $0.295/$1.20 per million tokens, it's cheaper than Gemini 2.5 Pro for input tokens. Best as a production alternative when you don't need fine-tuning — the base model already scores 87/100 which may be sufficient without training.

What Does Training Cost?

Training means we teach the model our database structure, table names, and Arabic language rules. It's a one-time payment — like a single orientation session.

Cost TypeWhat It MeansAmount
One-Time TrainingPay once to train with 500 examples$2.25 – $37.50
Per-Query CostAlready paid today, stays the sameSame as current
Server CostOnly if self-hosting Qwen 3 32B or MiniMax M2.5$0, ~$1,015 (Qwen), or ~$4,351+ (MiniMax)
💡 Key Point
The real investment is not money — it's the 2–3 weeks of effort to prepare 500 training examples (correct questions and answers in both English and Arabic). The compute cost is just $2–$38 regardless of which model you choose.
